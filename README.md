# Rock-Paper-Scissors-Lizard-Spock (python)

## Description
Small project from school which serves as an extension to the traditional game "Rock-Paper-Scissors".
the player competes against a computer and i added the elements "lizard" and "spock" from the Tv show "The Big Bang theory".

## Features
- player against computer game
- determine winner based on extended rules
- input validation for player choice
- display of computer and player choices
- result announcement ( e.g "Player wins")

## Technologies
- Pythons 2.x ( using 'raw_input')
- Standard library: 'random'
- Logic and code adapted from online tutorials for learning purposes

  ## How to Run
  1. Install Python2x or use a web-based Python environment
  2. Download the file 'rpsls.py'
  3. open terminal or command prompt
  4. run the program
  5. enter your choice, e.g 'rock'
